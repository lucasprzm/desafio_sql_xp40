# Desafio Hands On - E-commerce - em grupo

Olá Hackers!
Muito bem! Agora é a hora mergulhar no Back End!

Como vocês já são quase experts em lógica de programação, vamos começar a trabalhar em trios para fazer a nossa primeira estrutura de banco de dados em SQL! Agora em grupos, teremos ainda mais energia para poder concluir esse Hands On Week.

Temos um cenário de Comércio Eletrônico:

## Começando...

Para iniciar, precisamos no MySQL instalado na máquina, para assim realizarmos as consultas e utilização das querys.

## Desenvolvendo...

* git clone https://github.com/rapharodrigues04/desafio_sql_xp40
* Abra no Workbench.
* Dentro do projeto existe um arquivo com o nome de `criacao-tabelas.sql` que é onde está construída a extrutura de todas as tabelas.
* Também dentro do projeto, está um arquivo com o nome `1-exercicio.sql` onde estão as informações que utilizaremos para popular as tabelas criadas no tópico anterior.
* Seguindo esse passo-a-passo o sucesso é certo, Confia! :wave:
